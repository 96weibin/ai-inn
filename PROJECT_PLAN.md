## 📋 项目概述

AI-Inn 是一款面向民宿行业的智能管理系统，核心特点是通过 AI 助手实现自然语言交互，让房东可以随时随地通过语音或文字指令完成房间管理、入住办理等操作。

### 核心价值

- 🤖 **AI 智能助手** - 自然语言交互，随时随地管理民宿
- 📊 **多视图展示** - 表格、楼层图、3D视图等多种展示方式
- 🌐 **多平台对接** - 支持 Booking、Airbnb、美团等平台订单
- 💰 **智能排房** - 自动分析最优房间安排方案
- 📱 **移动友好** - 支持手机端操作

## 🛠️ 技术栈详情

### 前端
- **框架**：React 18 + TypeScript
- **构建工具**：Vite
- **UI 库**：Ant Design
- **路由**：React Router（可选）
- **状态管理**：React Hooks（初期）

### 后端
- **框架**：FastAPI
- **语言**：Python 3.14
- **ORM**：SQLAlchemy
- **数据库**：SQL Server
- **迁移工具**：Alembic

### 数据库
- **类型**：SQL Server
- **连接方式**：Windows 认证
- **表数量**：6 张核心表

### AI 模块
- **方案**：自研对话系统（初期）
- **意图识别**：基于规则 + 机器学习
- **语音支持**：可选功能（后期）

### 问题记录

#### 问题 1：图标库版本兼容性
- **问题**：@ant-design/icons v5 图标名称导入错误
- **原因**：图标导出方式变更
- **解决方案**：暂时移除图标，使用文字替代
- **状态**：✅ 已解决

#### 问题 2：GitHub SSH 连接失败
- **问题**：Deploy Key 权限不足
- **原因**：密钥只授权给了 MyNotes 仓库
- **解决方案**：为 ai-inn 仓库添加新的 Deploy Key
- **状态**：✅ 已解决
